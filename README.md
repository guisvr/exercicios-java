# Exercícios Java - Programação Orientada a Objetos

Este repositório armazena meus estudos práticos em Java, focando em conceitos de POO e manipulação de coleções.

## Projeto: Sistema de Gerenciamento de Estoque

Desenvolvi uma aplicação para controle de produtos que simula operações reais de uma loja. O foco principal foi estruturar o código de forma organizada e segura.

### Implementações principais:
- **Encapsulamento**: Uso de modificadores de acesso `private` e métodos `getters` para garantir a integridade dos dados dos produtos.
- **Manipulação de Listas**: Implementação de `ArrayList` para gerenciar dinamicamente o estoque.
- **Cálculos Automáticos**: Métodos para reajuste de preço em massa e cálculo do valor total do patrimônio (Preço x Quantidade).

## Como executar
Basta compilar os arquivos na pasta `src` e rodar a classe `Principal.java`.

Neste projeto, o meu maior foco foi entender como manipular objetos dentro de uma lista. Foi um exercício importante para consolidar o uso de ArrayList em vez de arrays fixos, o que permitiu criar uma lógica de reajuste de preços que percorre todo o estoque de forma dinâmica. Também aproveitei para praticar o fluxo de trabalho com Git, aprendendo a lidar com versões e organização de pastas diretamente pelo terminal.
